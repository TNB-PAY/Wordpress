TNBPAY is a free and open-source Woo-commerce plugin which allows you to accept TNBC without fees or intermediaries.


## Features

* No transaction fees (other than the network fee)
* Creates TNBC as a currency on installation in Woo-commerce
* Works with both USD and TNBC as currency.
* No fees, middleman or Escrow
* Secured (No request for Signing/Private key)


## Getting Started
Download the plugin as a zip file [here](https://github.com/akandejaphet/TNBPay/archive/refs/tags/1.1.0.zip) then install on Wordpress. Please check [Documentation](https://github.com/akandejaphet/TNBPay/blob/master/Documentation/TNBPAY%20Documentation.pdf) for full installation procedure
